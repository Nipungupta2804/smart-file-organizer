# 📂 Smart File Organizer

A Python-based smart file management system that automatically organizes files, detects duplicates, and maintains a clean workspace.

---

## 🚀 Features

- 📁 Automatic file organization
- 🧠 Duplicate detection using hashing
- 🗑️ Safe Trash system (no direct deletion)
- 🏷️ Smart file renaming
- 🌍 Cross-platform support (Windows, Mac, Linux)
- 🖥️ Streamlit-based UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- os, shutil
- hashlib
- datetime

---

## ⚙️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
